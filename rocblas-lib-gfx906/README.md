# Tensile / rocBLAS Kernel Artifacts

This repository contains precompiled GEMM kernel artifacts generated by
AMD’s open-source Tensile and rocBLAS projects. These files are part of the
ROCm linear-algebra stack and are normally distributed inside the official
ROCm packages under:

    /opt/rocm/lib/rocblas/library/

The files include:
- `.hsaco` — final HSA GPU code objects for a specific architecture
- `.co`    — intermediate code objects used during kernel generation
- `.dat`   — metadata files describing kernel parameters and tuning results

All of these artifacts originate from the open-source Tensile kernel
generator and the rocBLAS build system. Both projects are licensed under
the MIT License, which permits redistribution and reuse as long as the
license notice is preserved.

These files are provided here for convenience, reproducibility, and use in
custom ROCm or GPU-compute builds targeting the listed architectures.
